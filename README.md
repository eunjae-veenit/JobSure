# 🚀 JobSure – Job Portal & Career Development Platform

JobSure is a comprehensive PHP and MySQL-based job portal designed to connect students and job seekers with employers. The platform offers job listings, internship opportunities, skill assessments, resume uploads, certificate generation, password recovery via OTP, and an administrative dashboard.

---

## 🌟 Features

### 👤 User Features
- User Registration and Login
- Secure Password Reset using OTP Verification
- Profile Management
- Resume Upload
- Job Search and Filtering
- Apply for Jobs and Internships
- Bookmark Favorite Jobs
- Career Guidance Courses
- Skill Assessments and Result Analysis
- Certificate Generation in PDF Format
- Chat Support

### 🛠️ Admin Features
- Admin Dashboard
- Add, Edit, and Delete Job Listings
- Manage User Applications
- Toggle Job Bookmarks
- Monitor Platform Activity

### 📄 Additional Features
- PDF Certificate Generation using FPDF
- Email Sending using PHPMailer
- OTP-based Authentication
- File Upload Management
- Responsive User Interface
